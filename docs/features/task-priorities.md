# Task Priorities (Frontend)

## Goal
Display and manage task priority in UI.

---

## Backend Already Exists
- critical
- high
- medium

---

## UI Requirements

### Task List
- Show priority badge per task
- Colors:
  - critical → red
  - high → orange
  - medium → blue
  - low → gray

---

### Create/Edit Task
- Dropdown selector for priority
- Default = medium

---

## Behavior
- Priority should be visible everywhere tasks are shown
- No redesign needed, just enhancement
